Projetos

Santander API RESTful

Este projeto consiste em uma API RESTful desenvolvida em Java 17 com Spring Boot 3, utilizando Spring Data JPA para integração com bancos de dados SQL. A documentação da API é gerada automaticamente com Swagger (OpenAPI), facilitando o entendimento e uso da API. O projeto foi implantado na plataforma Railway, que oferece deploy em nuvem, monitoramento e integração contínua (CI/CD).

Quiz "The Last of Us"

Este projeto é um quiz interativo baseado em The Last of Us, onde o usuário descobre qual personagem do jogo mais se parece com ele. Desenvolvido com HTML, CSS e JavaScript, oferece uma experiência simples e divertida.

viaCep - REST API

API REST construída em Java com Spring Boot, que utiliza a ViaCEP API para buscar e gerenciar informações de endereço com base no CEP fornecido. Permite operações CRUD para cadastrar e consultar clientes.
